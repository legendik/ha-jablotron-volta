1. Otevři Home Assistant
2. Settings → Dashboards → + ADD DASHBOARD
3. Zadej název: "Jablotron Volta"
4. Edit Dashboard → ⋮ → Raw configuration editor
5. Zkopíruj obsah dashboard_jablotron_volta.yaml
6. SAVE
7. Hotovo! 🎉
